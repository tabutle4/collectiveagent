## Features

✅ **Professional Layout:** Uses your luxury email template  
✅ **User Info:** Shows who sent the message  
✅ **Message Box:** Gray background with gold accent border  
✅ **Line Breaks:** Preserves formatting from textarea  
✅ **Auto CC:** User gets a copy of their message  
✅ **Branded:** Matches all other emails from the system

---

## Email Flow

1. User clicks "Contact" in menu
2. Types message in modal
3. Clicks "Send"
4. Email sent to: office@collectiverealtyco.com
5. Email CC'd to: user's email
6. User sees confirmation
7. Office receives support request
